Key Name:
ActivateModifier

Possible Value(s):
Shift, NumLock, Ctrl, Alt, Super

Description:
Sets the activate modifier key which is pressed in conjunction with the key set
for the ActivateKey setting in order to enable/disable xkeycursor.
